# La P'tite Clarine — Site web

Site vitrine du restaurant savoyard **La P'tite Clarine**, situé à Samoëns (74).

## Structure

- `index.html` — Page principale
- `style.css` — Styles
- `script.js` — Interactions (parallax, menu, animations)
- `menu-data.js` — **Données du menu** (modifier ici les plats et prix)

## Modifier le menu

Ouvrir `menu-data.js` et modifier les noms, descriptions et prix. Aucune connaissance technique requise.

## Lancer en local

Ouvrir `index.html` dans un navigateur, ou :

```bash
npx serve .
```
